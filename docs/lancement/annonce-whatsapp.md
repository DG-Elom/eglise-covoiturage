✅ Gratuit, simple, communautaire
✅ Trouve un trajet près de chez toi

## Texte WhatsApp (version longue)

```
🚗 *Covoiturage ICC Metz — c'est en ligne !*

Famille ICC, j'ai le plaisir de vous présenter notre nouvelle app de covoiturage entre fidèles 🙌

👉 https://icc-covoit.fr

*Pour qui ?*
• Les fidèles motorisés qui veulent partager leurs trajets vers le culte
• Les fidèles sans voiture qui cherchent un moyen de venir
• Toute personne qui souhaite renforcer la communion fraternelle 💚

*Comment ça marche ?*
1. Ouvre le lien icc-covoit.fr sur ton téléphone
2. Crée ton compte (Google ou email)
3. Conducteur : propose un trajet
4. Passager : recherche un trajet près de chez toi
5. L'app vous met automatiquement en contact

*Astuce iPhone* : sur Safari, clique le bouton "Partager" 📤 puis "Sur l'écran d'accueil". L'icône apparaît comme une vraie app.

*Astuce Android* : dans Chrome, clique le menu ⋮ puis "Ajouter à l'écran d'accueil".

*C'est gratuit et 100% communautaire* — pas de pub, pas d'argent, juste l'esprit d'entraide.

Pour toute question : rgpd@icc-covoit.fr

Que Dieu nous bénisse dans cette initiative ! 🙏
```

## Texte court pour annonce en culte / podium

```
Frères et sœurs, l'application "Covoiturage ICC Metz" est désormais disponible.
Elle permet aux fidèles motorisés et non motorisés de s'organiser pour venir au culte ensemble.
Rendez-vous sur icc-covoit.fr ou flashez le QR code à l'accueil.
```

## QR Code

Le QR code pointant vers https://icc-covoit.fr est dans ce dossier :
`qr-code-icc-covoit.png` (600×600 px, niveau de correction H = haute résistance).

À utiliser pour :
- Flyers / affiches à imprimer
- Slide à projeter avant/après le culte
- Bannière dans le hall d'entrée de l'église

## Suggestions visuelles

Pour un flyer A6/A5, suggestion de mise en page :
- En-tête : logo ICC + "Covoiturage ICC Metz"
- Centre : QR code (400×400 px imprimé)
- Sous le QR : `icc-covoit.fr` en grand
- Footer : "Allez à l'église ensemble. Gratuit, simple, communautaire."
